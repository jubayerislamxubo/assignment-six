# DigiTools Shop - Assignment 6

এটি আমার ৬ষ্ঠ অ্যাসাইনমেন্টের একটি প্রজেক্ট, যেখানে আমি রিঅ্যাক্ট (React) ব্যবহার করে একটি ডিজিটাল টুলস শপ তৈরি করেছি। এই প্রজেক্টের মূল উদ্দেশ্য ছিল ডাইনামিক কম্পোনেন্ট এবং স্টেট ম্যানেজমেন্টের মাধ্যমে একটি ইউজার-ফ্রেন্ডলি ইন্টারফেস তৈরি করা।

## 🛠️ এই প্রজেক্টে যা যা করা হয়েছে:
- **ডাইনামিক প্রোডাক্ট কার্ড:** এখানে ৬টি আলাদা আলাদা ডিজিটাল টুলসের কার্ড ডিজাইন করা হয়েছে।
- **কার্ট ফাংশনালিটি:** ইউজাররা সহজেই প্রোডাক্ট কার্টে যোগ করতে এবং প্রয়োজন অনুযায়ী রিমুভ করতে পারবেন।
- **রেসপন্সিভ ডিজাইন:** টেলউইন্ড সিএসএস (Tailwind CSS) ব্যবহার করে সাইটটিকে মোবাইল এবং ডেস্কটপ—উভয় ডিভাইসের জন্য উপযোগী করা হয়েছে।
- **ইন্টারেক্টিভ ইউআই:** সুন্দর হোভার ইফেক্ট এবং আইকন ব্যবহার করে ইউজার এক্সপেরিয়েন্স উন্নত করার চেষ্টা করেছি।
